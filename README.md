# Pøllebrors 35-års Quest

En lille statisk GitHub Pages-side med et retro arcade-spil, der afslører en gave med koncertbilletter.

## Sådan slår du GitHub Pages til

1. Gå til repositoryets `Settings` > `Pages`.
2. Vælg `Deploy from a branch`.
3. Vælg `main` og `/root`.
4. Gem. GitHub viser linket, når siden er klar.

## Tilpas gavebeskeden

Finaleteksten ligger i `index.html` i sektionen med `id="finalScreen"`.
